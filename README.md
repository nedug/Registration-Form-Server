# Server for MERN User Registration Form

***

* User registration on MongoDB
* User Login + JWT Token
* User Profile
* Create notes
* Change password
* Remove user

***

```
Technologies and libraries in development:
- Node.js
- Express
- MongoDB + mongoose
- jsonwebtoken + bcryptjs
- nodemon
```
